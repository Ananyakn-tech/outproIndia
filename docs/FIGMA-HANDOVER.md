# Figma Handover Checklist

This document lists the artboards and assets to include when delivering Figma files to the client.

- Include editable Figma files for: Home, Services, Service Detail, About, Contact
- Provide component library (Buttons, Forms, Typography, Colors, Spacing)
- Export assets: SVG icons, logo files (SVG + PNG), hero illustrations (PNG at 2x)
- Include a pages index with notes explaining layout constraints and responsive breakpoints.
- Attach a PDF export of the Figma file for non-design stakeholders.

Notes: I cannot create Figma files from code. Please upload the Figma files to a shared link and attach them to the project handover.
